Question-1: What are the different ways to select an element in the DOM?

The most commonly used methods of DOM include getElementById, which allows to select an element by its unique ID, and getElementsByClassName or getElementsByTagName, which allow to select elements based on their class or tag name, respectively. For more complex selections, querySelector and querySelectorAll are powerful tools that allow to use CSS selectors to target elements.

Question-2: What is the difference between innerHTML, innerText, and textContent

InnerHTML is used to get or set the HTML markup inside an element, including any tags,It allows to work with complex HTML structures. On the other hand, innerText retrieves the visible text content, ignoring HTML tags but taking into account how the content is rendered on the page. textContent, while similar to innerText, differs in that it retrieves all text content, including text inside hidden elements, It does not consider CSS visibility.

Question-3: What is event delegation in the DOM?

It is a method where instead of adding separate event handlers to multiple elements an event handler is added to a parent element. When an event occurs on the parent element, it "bubbles" down to the child elements, where the necessary action is performed. This method improves performance and works without having to write separate event handlers for new elements when they are added.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where a DOM event first occurs on the target element and then "bubbles" up to the parent elements above, meaning the event starts at its parent element and reaches the elements above it.

Question-5: How do you create, add, and remove elements using JavaScript?

To create a new element, we use the document.createElement() method, which creates an HTML element of a specific type. After creating the element, you can add it to the DOM using methods like appendChild(), which associates the element with a specific parent. To remove an element, use removeChild() on the parent element or remove() directly on the element to remove it from the DOM.
